# ♜ BlitzClock

A modern chess clock built with SwiftUI.

## Features
- Dual player countdown timer
- Tap-to-switch turns
- Fischer increment support
- Custom time controls (including asymmetric time)
- Sound feedback (with mute option)
- Move counter for both players
- Haptic feedback alerts
- Low-time warning indication
- Customizable clock colors
- Customizable clock colors
